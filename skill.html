스킬

스킬이란?

S,A,B,C,D급으로 나뉘어져 있으며 사용자에게 엄청난 이득을 주는 아이템입니다.

사용법은 F키 입니다 ( 손 바꾸기 키 )

아래에는 스킬의 설명이 있습니다.

S급 목록

빛의 축복 ( S )

빛이 주변의 적에게 디버프 또는 주변의 아군에게 버프를 해줍니다.
쿨타임 : 25초

전면 폭격 요청 ( S )

4번의 폭격을 요청합니다.
데미지 : 100 * 4
쿨타임 : 15초

파멸의 검기 ( S )

돌진해, 돌진한거리에 있는 모든 적을 썰어버립니다.
데미지 : 2 * 100
쿨타임 : 5초

이펙트 : 포화 , 힘
적용대상 : 자신
쿨타임 : 72초

쿨타임 : 17초

암살 ( A )

쿨타임 : 9초

벽력일섬 ( A )

쿨타임 : 16초
