홈으로 서버 기본 설명 스킬 명령어

명령어 가이드

/메뉴

  • Shift + F 키 조합으로도 메뉴를 열 수 있습니다.
  • 다양한 명령어를 클릭을 통해 편리하게 실행 가능합니다.

/파티

  • 생성 : 파티를 새롭게 생성합니다.
  • 초대 <플레이어> : 지정한 <플레이어>를 파티에 초대합니다.
  • 수락 : 자신에게 도착한 파티 초대를 수락합니다.
  • 정보 : 현재 파티에 소속된 유저 목록을 확인합니다.
  • 채팅 : 파티 전용 채팅 모드를 ON/OFF 합니다.
  • 탈퇴 : 현재 소속된 파티에서 나갑니다.
  • 해산 : 파티를 완전히 없앱니다. (파티장 전용 기능)

/로비

  • 서버의 스폰 구역(로비)으로 즉시 이동합니다.
  • 동일한 기능의 단축 명령어: /l, /lobby, /spawn, /넴주

/칭호

  • 획득한 칭호를 선택하여 캐릭터에 장착할 수 있는 전용 보관함을 엽니다.

/sethome, /home

  • sethome <숫자> : 현재 유저의 좌표 위치를 지정한 번호(<숫자>)의 홈으로 저장합니다. (이미 존재할 시 덮어쓰기)
  • home <숫자> : 저장해 둔 해당 번호(<숫자>)의 홈 위치로 즉시 텔레포트합니다.

/강화

  • 장비 및 아이템 강화 전용 UI 창을 출력합니다.
  • 이를 통해 무기, 장비, 방어구의 능력치를 올릴 수 있습니다. (자세한 내용은 강화 탭 참고)

/직업

  • 직업을 아직 선택하지 않은 상태라면 직업 선택 가이드 UI가 출력됩니다.
  • 이미 전직을 완료한 상태라면 직업 상세 정보창이 오픈됩니다.
  • 직업 창 내부에서는 현재 보유한 경험치, 수행할 임무, 다음 레벨 도달 시 획득할 스텟을 볼 수 있습니다.

/town (마을 서버에서만 작동)

  • create : 현재 플레이어가 서 있는 자리를 기점으로 새로운 마을을 건설합니다.
  • claim : 현재 자신이 서 있는 구역을 마을의 소유 영토로 편입(점유)합니다.
  • declaim : 현재 서 있는 위치에 확보된 마을 소유 영토의 점유를 해제하고 판매합니다.
  • set
    • spawn : 현재 발을 딛고 있는 위치를 우리 마을의 공식 스폰 지점으로 지정합니다.
    • taxes <숫자> : 마을 구성원들에게 부과할 세금 비율을 <숫자>% 만큼 설정합니다.
  • invite <플레이어> : 특정 플레이어를 우리 마을의 구성원으로 초대합니다.
  • join : 나에게 온 마을 가입 초대를 받아들여 해당 마을의 일원이 됩니다.
  • leave : 현재 소속되어 있는 마을에서 자진 탈퇴합니다.
  • kick <플레이어> : 영지 내 부적절한 플레이어를 마을에서 강제 추방합니다.
  • list : 생성되어 있는 전체 마을 목록이나 해당 마을의 요약 정보를 출력합니다.
  • spawn : 소속된 우리 마을의 지정 스폰 지점으로 즉시 순간이동합니다.
